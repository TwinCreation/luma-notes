Our Privacy Philosophy

Your privacy is important to us. This Notes app is designed to collect only the minimum data required to provide its core features. We do not sell, rent, or misuse your personal information.

Permissions and Their Usage

1. Notifications Access

  • Permission: android.permission.POST_NOTIFICATIONS (Android 13+)
  • To send reminders and important notifications
  • To alert you about scheduled tasks or updates

2. Internet Access

  • Permission: android.permission.INTERNET
  • To load privacy policy and terms pages
  • Optional network-based features (if applicable)

3. Media Access

  • READ_MEDIA_IMAGES / VIDEO / AUDIO
  • To attach media files to notes

4. Storage Access (Legacy)

  • READ_EXTERNAL_STORAGE (Android 12 and below)
  • For compatibility with older devices

File Sharing and Data Handling

  • Secure FileProvider URIs for safe file sharing
  • Storage Access Framework (SAF) for controlled file selection
  • Persistable permissions for long-term access

Data Storage

  • All data stored locally on device
  • No cloud storage or external servers
  • No third-party sharing

Data Security

We implement appropriate technical measures using secure Android system APIs to protect your data.
